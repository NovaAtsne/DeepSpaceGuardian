## DeepSpaceGuardian 是什么？

简单来说，DeepSpaceGuardian是一款RGB小游戏，使用Python+Pygame语言编写，其前身为Dodge Game，目前DeepSpaceGuardian已经发行到1.0版本。对应着Dodge Game 1.6版本。

## 如何下载DeepSpaceGuardian？

很简单，你可以到[这篇文章](/download.md)中下载你需要的DeepSpaceGuardian。同样，你也可以在DeepSpaceGuardian的[Github](https://github.com/NovaAtsne/DeepSpaceGuardian/releases)上下载到游戏
